# Arrow Game 

Petit jeu HTML/CSS/JS où vous tirez des flèches avec un arc.

## Description

Fichier principal : `arrow-game.html`.
Le jeu utilise SVG et GSAP (TweenMax + MorphSVGPlugin) pour animer l'arc et les flèches. Il suffit d'ouvrir `arrow-game.html` dans un navigateur moderne pour jouer.

## Fichiers

- `arrow-game.html` — le jeu (HTML + CSS + JS inline)
- `README.md` — ce fichier

## Comment lancer

Double-cliquez sur `arrow-game.html` ou ouvrez-le depuis votre navigateur. Sous Windows PowerShell :

```powershell
Start-Process -FilePath 'c:\Users\PC NANO\OneDrive\Bureau\lina\arc\arrow-game.html'
```
